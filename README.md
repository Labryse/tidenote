# TideNote

A modern note-taking and whiteboard app. Write, draw, and organize — all in one place.

**Web:** https://tidenote-22fde.web.app  
**Download:** Available for Windows (Mac, Linux, Android, iOS coming soon)

---

## About

TideNote combines a block-based document editor with an infinite canvas whiteboard. Built for people who think in both text and visuals.

- Block editor with rich content types
- Infinite canvas with shapes, diagrams, and sticky notes
- Real-time cloud sync across devices
- Offline support
- Available on Web, Windows, and Android
